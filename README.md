# Solfoundary Assignment

A multi-step onboarding flow built with React, TypeScript, and Tailwind CSS.

## Tech Stack

- **React 19** with TypeScript
- **Vite** for development and bundling
- **Tailwind CSS v4** for styling
- **ESLint** for code linting

## Project Structure

```
src/
├── components/
│   ├── icons/          # Custom SVG icon components
│   ├── modules/
│   │   └── onboarding/ # Onboarding flow (cards, insights, segments, etc.)
│   └── ui/             # Reusable UI components (Badge, Button, Card)
├── data/
│   └── onboardingConfig.ts  # Step configuration data
├── App.tsx
├── main.tsx
└── index.css
```

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

### Linting

```bash
npm run lint
```
